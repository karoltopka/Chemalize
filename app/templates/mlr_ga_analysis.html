Step 3: Standardization and Variance Check
Standardize features to mean=0 and variance=1
Remove features with zero variance (constant values)
Remove features with variance below threshold
Features with variance below this threshold will be removed
Internal Cross-Validation Type

Select the cross-validation method used for fitness evaluation (R²cv) during GA optimization.

K-Fold CV: Standard k-fold cross-validation. Data is split into k equal folds, and each fold is used once as validation while the remaining folds are used for training.

Number of folds can be configured in Step 4 (GA Parameters).

Sorted Stratified CV: Data is sorted by Y values, then every n-th element goes to the test set. Multiple iterations are performed with different starting offsets. This ensures representative sampling across the full range of Y values.
e.g., 5 means ~20% of data goes to test set per iteration. Number of iterations is set in Step 4.
