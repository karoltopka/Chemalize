# ChemAlize

A web-based platform for chemometrics and chemical data analysis, available at **[TopKarol.pl](https://topkarol.pl)**.

## What you can do

**Data**
- Upload datasets (CSV, XLSX, JSON, YAML, TXT)
- Preprocess data — scaling, normalization, column selection, outlier removal
- Merge multiple datasets into a unified file

**Analysis**
- **PCA** — Principal Component Analysis with interactive score and loading plots
- **MLR** — Multiple Linear Regression with variable selection
- **MLR + GA** — MLR with Genetic Algorithm for automated variable selection
- **PCR** — Principal Component Regression
- **Clustering** — k-means, hierarchical, and other clustering methods
- **Classification** — Decision Tree, KNN, Logistic Regression, Naive Bayes, Random Forest, Linear SVC

**Specialized modules**
- **Alvadesk** — Analysis of molecular descriptors from Alvadesk output
- **EPI Suite** — Applicability Domain assessment for EPI Suite models (BCFBAF, BioWin, Kocwin, Kowwin, MPBPWIN)
- **NanoTox** — TiO2 nanoparticle toxicity prediction using Explainable Boosting Machines
- **ScopeHub** — Scientific database search query generator (PubMed, Scopus, Web of Science, IEEE and more)

**Export**
- Download plots, PC scores, results as CSV
- PDF report generation

## Built with
